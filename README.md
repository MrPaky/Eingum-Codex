# Eingum Codex

*A fantasy design framework for Obsidian inspired by tabletop RPG sourcebooks.*

## Philosophy

Eingum Codex is not just a theme.

It is a visual language designed to make worldbuilding feel like reading an ancient fantasy codex.

Every component has a narrative purpose before it has a graphical one.

## Design Principles

* Story before decoration.
* Every visual element behaves like what it represents.
* Knowledge is layered, never dumped.
* Empty space is part of the design.
* Consistency creates immersion.
